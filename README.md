## ThoughtWorks Docker Image: redis

[![](http://dockeri.co/image/baselibrary/redis)](https://registry.hub.docker.com/u/baselibrary/redis/)

### Base Docker Image

* `latest`: redis 3.0
* `3.0`   : redis 3.0
* `2.8`   : redis 2.8

### Installation

    docker pull baselibrary/redis

### Usage

    docker run -it --rm baselibrary/redis
